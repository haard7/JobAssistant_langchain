# Job Assistant Agent (Langchain, Streamlit and OpenAI)

### Match your Resume with Job Description and get exactly what you need
- For personalize message to HR
- For Exact skill to put in Resume
- and Even Sample resume for perfect candidate for that job


## how to Run 

1. clone repo
2. create virtual env in python
3. installl required dependencies
    `pip install -r requirements.txt`
4. run `streamlit run jobassistant.py`
